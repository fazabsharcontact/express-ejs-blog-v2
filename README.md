# Express Blog App V2

A blog web application built with **Node.js**, **Express.js**, **EJS**, and **Tailwind CSS**.

This project is an improved version of a basic blog application, featuring full CRUD functionality, search, pagination, detail pages, and data persistence using JSON files.

---

## Features

- Create, Read, Update, Delete (CRUD) blog posts
- View post details on a dedicated page
- Search posts by title or content
- Pagination for better content navigation
- Data persistence using JSON files (no database)
- Server-side rendering with EJS
- Reusable layout using EJS partials
- Clean and responsive UI with Tailwind CSS

---

## Tech Stack

- Node.js
- Express.js
- EJS
- Tailwind CSS (CDN)
- File System (JSON storage)
